## 학습된 교훈
- FE(페어와이즈 조합, TargetEncoder)는 단독 XGBoost에서 효과 없음
- 학습 설정 개선(10-Fold, 낮은 lr)이 가장 확실한 개선
- OOF 기반 가중치 최적화는 과적합 위험 → 단순 평균이 더 안전할 수 있음
- LightGBM이 현재 설정에서 부진 → 튜닝 필요
